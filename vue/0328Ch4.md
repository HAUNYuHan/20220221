# 事件處理
## v-on
```

<!DOCTYPE html>
<html lang="en">

<head>
  
</head>

<body>

	<div id="app">
		<p>Cont:{{ count}}</p>
		<button v-on:click="count++">Plus</button>
	</div>

<script src="https://unpkg.com/vue@next"></script>
	<script>
		const vm = Vue.createApp({
			data () {
				return{
					count:0
				}
			},
		}).mount('#app');
	</script>
 </body>    
</html>
```
