## 屬性綁定 v-bind
### ex
```
<!DOCTYPE html>
<html lang="en">
 <head>

 </head>
<div id="app">
	<p v-bind:id="customId"> Show time!</p>
</div>

<script>
	const vm = Vue.creatApp({
		data()	{
			return	{
				customId:'item-id-l'
			}
		}
	}).mount('#app')
</script>
</body>    	
</html>

```
