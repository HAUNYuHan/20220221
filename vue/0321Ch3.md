## 屬性綁定 v-bind
### ex
```
<!DOCTYPE html>
<html lang="en">
 <head>

 </head>
<div id="app">
	<p v-bind:id="customId"> Show time!</p>
</div>

<script>
	const vm = Vue.creatApp({
		data()	{
			return	{
				customId:'item-id-l'
			}
		}
	}).mount('#app')
</script>
</body>    	
</html>

```
## 表單綁定v-mode
```
<!DOCTYPE html>
<html lang="en">
 <head>

 </head>
 <body>
<div id="app">
<input v-model="message" placeholder = "edit me">
<p>Message is: {{message}}</p>
	
</div>

	<script src="https://unpkg.com/vue@next"></script>
<script>
   const vm = Vue.createApp({
    data() {
	  return {
		message: 'hi,ksu!'
	  }
	 },
	}).mount('#app')
  </script>
 </body>    	
</html>
```
